# Estimacion-tabla-de-mortalidad
Este proyecto aborda la estimación de la tasa de mortalidad ($q_x$) en una población real (2006-2015) sujeta a censura tipo I y retiros. Se implementaron técnicas de suavizamiento mediante regresión ponderada por núcleo (Kernel) , analizando el impacto del balance sesgo-varianza al ajustar el ancho de banda.
